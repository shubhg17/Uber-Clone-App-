# 🚖 Uber Clone (Frontend)

A modern and responsive **Uber Clone (Frontend)** built using React and Tailwind CSS. This project focuses on practicing frontend development by replicating core UI/UX features of a ride-booking application.

---

##  🚀 Features

* Clean and responsive UI design
* Multiple pages (Home, Ride Selection, etc.)
* Component-based architecture using React
* Styled using Tailwind CSS
* Interactive UI elements (buttons, inputs, cards)
* Smooth layout and positioning using Flexbox & Grid
* Optimized for fast performance

##  🔧 How to Run

Clone the repository:

```
git clone https://github.com/your-username/uber-clone.git
```

Navigate into the project folder:

```
cd uber-clone
```

Install dependencies:

```
npm install
```

Start the development server:

```
npm run dev
```

* Open the given localhost URL in your browser.
